## Environment

Node version: ~16.x.x

## Install

`npm install`

---

### Usage in Dev

`npm start`

---

## Build

`npm run build`

Result make **build** folder contain application.

### Test build

Install **serve** package in global scope:

```bash
npm install --global serve
```

Then, serve a static site/single page application:

```bash 
serve -s build
```
